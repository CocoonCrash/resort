# Resort - WebCam over IPFS 

## Install & Run

- `npm install`
- `npm run dev`
- Open a webserver in the root directory
- Navigate to page and accept sharing webcam
- Open a new incognito window with same page
- Enjoy seeing yourself, TWICE!
